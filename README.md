#Carousel Framework

Carousel.js 是一个轻量级脚本触摸滑动类库。

轮播（Carousel）插件是一种灵活的响应式的向站点添加滑块的方式。除此之外，内容也是足够灵活的，可以是图像、视频或者其他您想要放置的任何类型的内容。
